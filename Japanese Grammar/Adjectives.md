All adjectives fall under two categories: [[na-adjective]] and [[i-adjective]].
## Exception

### Conjugation for 「いい」

|              | Positive | Negative |
| ------------ | -------- | -------- |
| **Non-Past** | いい       | よくない     |
| **Past**     | よかった     | よくなかった   |
### Conjugation for 「かっこいい」
|              | Positive | Negative  |
| ------------ | -------- | --------- |
| **Non-Past** | かっこいい    | かっこよくない   |
| **Past**     | かっこよかった  | かっこよくなかった |
